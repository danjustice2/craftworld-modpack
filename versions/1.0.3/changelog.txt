# Server Update

## Changes and Updates

### General Updates
- **Updated server sqlite file** for Distant Horizons.
- **Added more splash texts** to enhance the user experience.

### Datapack Management
- **Removed broken datapacks** and replaced them with new ones.
- Added support for **more foods in the Diet mod**.

### Distant Horizons
- **Downgraded Distant Horizons** from the experimental version with server support due to issues with lag and shader skyboxes.
  - Distant Horizons will still be supported through gradual releases of updated sqlite files as needed.
- Added **delete_dh_server_cache.bat** to assist in deleting Distant Horizons server caches, which can sometimes help fix issues.

### Configuration Sync
- **Synced config files** from the server, fixing various bugs and providing a more consistent experience when playing in single-player mode.
